This page provides a cross-reference of all catalog types organized by their parent entity.

## Device

| Name | Description |
| ---- | ----------- |
| [DeviceType](../devices/types.md#type-devicetype) | A classification type for devices. |
| [DeviceStatus](../devices/types.md#type-devicestatus) | An operational status for devices. |
| [DeviceModel](../devices/types.md#type-devicemodel) | A specific device model produced by a vendor. |
| [DeviceVendor](../devices/types.md#type-devicevendor) | A device manufacturer or vendor. |
| [DeviceRelationType](../devices/types.md#type-devicerelationtype) | A type of relationship between two devices. |
